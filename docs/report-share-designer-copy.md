幕标：

```text
Act I · 喜剧场
```

场景云：

```text
理想主义发布会 vs 直播间的“交个朋友”。
同一张嘴，两种真相。
他拍拍自己的脸：
“打脸归打脸，活儿还得干。”
```

```text

```

```text
Act II · 正剧场
```

```text
“那你说说——什么叫败坏？什么叫虔敬？
```

```text
你们审判我，我审判问题本身。
最后一口气，也留给追问。
```

### 名人点评 · 罗永浩篇

标题：

```text
罗永浩自评：我这叫“现实驱动型系统更新”
```

正文：

```text
你们都说我打脸。行，我认。
但你们想想，一个人敢把几年前的发布会拿出来跟现在的自己对比，这事儿本身就挺赛博判官的。
我当初那些“天生骄傲”，立的标准比谁都高。后来现实啪啪打脸，怎么办？嘴硬没用，跑路更难看。那我就坐下来，把“以前我说的”和“现在我得做的”摆一桌，自己审自己。
判词很简单：脸可以肿，逻辑不能塌。打脸归打脸，更新完系统继续干活。
所以这个赛博判官类型，说白了就是——你敢不敢给自己开庭？
我开了，我判了，我还在干活。就这。
```

传播短句：

```text
罗永浩同款判官：打脸是版本更新，不是卸载重装。
```

### 名人点评 · 苏格拉底篇

标题：

```text
苏格拉底自评：我只是把审判场改成了讨论组
```

正文：

```text
他们说我快死了还在嘴硬。我想了想，说这是“求知”。
那天的情形很简单——五百个雅典人坐在那儿，准备判我。我看了看起诉书，觉得问题太多，得一个一个过。
“败坏青年”是什么意思？先定义。
“不信神”又指什么？再讨论。
我说我没时间害怕，因为你们给的题目里有太多没澄清的概念。
你看，这就是赛博判官。别人觉得你倔、你轴、你死到临头还在抬杠。但你自己知道，你不是在抬杠，你是在审判混乱本身。
毒酒可以喝，但问题必须先说清楚。
如果这个测试把我归进这个类型——我会先问问这类型的定义是什么，再决定要不要认。
```

传播短句：

```text
苏格拉底同款判官：死亡面前，先问定义，再谈情绪。
```

## 类型描述 · 有梗概括

标题：

```text
赛博判官：脑内自带审题庭的人间清醒
```

正文：

```text
你不是爱吵架，你是受不了一件事：
一群人明明可以把问题说清楚，偏偏用热闹掩盖标准的缺席。

所以你的脑子里常驻一个小型审题庭。
左边坐着不吹不黑，右边站着菜就多练。
任何模糊讨论进入，都会被拆成：
逻辑链、证据包、价值点、行动项。

罗永浩打脸后继续干活，
苏格拉底临死前还在追问定义——
这俩都是你，一个喜剧版，一个正剧版。
共同点只有一条：场面可以乱，逻辑不能乱；态度可以软，标准不能软。

你是赛博判官。
你负责在所有人都说“差不多得了”的时候，
把桌子一拍：先别走，这题还没判完。
```

## 调性调整备注

如需调整任一幕的严肃度、喜感度，或增加更多弹幕选项，可优先从以下三处微调：

- 舞台场景云：决定戏剧感和画面信息。
- 名人第一人称弹幕：决定传播梗密度。
- 右侧名人点评正文：决定类型解释深度。
