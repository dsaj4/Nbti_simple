
## V1 开麦嘴替版

版本代号：`copy-v1-open-mic-mouthpiece`

修改日期：2026-06-04

修改前基线：`dac655a Add report share copy workbench`

修改目标：

- 保留“公正感、锋利感、自洽感、传播感”。
- 增强梗浓度、网感和群聊截图传播感。
- 让文案更像“赛博判官替我开麦”，而不是产品在解释人格。

修改范围：

- 页面浏览器标题与描述。
- 页面顶部副标题和主操作按钮。
- 左侧舞台的默认态、罗永浩幕、苏格拉底幕：场景云、问句、回答气泡。
- 右侧类型说明、赛博判官自评、名人点评、短判词。
- 分享文案池和版本按钮。
- 拉绳、切幕、复制结果等微交互文案。

保留原则：

- 不改变页面结构。
- 不改变“场景云 = 出题 / 演员 = 演出 / 回答气泡 = 说台词”的核心模型。
- 不新增真实背书语气，名人内容仍保持拟态点评和戏剧化解释。
- 不把赛博判官写成单纯攻击型角色，仍保留“标准、证据、定义”的正当性。

回退方式：

```bash
git checkout copy-v1-open-mic-mouthpiece
```

如需回到本版之前，回退到修改前基线 commit：

```bash
git checkout dac655a
```

验证方式：

- 打开报告分享页确认默认态、罗永浩幕、苏格拉底幕文案同步切换。
- 截图保存 `screenshots/report-share-page.png`。
- 检查分享文案按钮是否能切换为标准版、开庭版、名人版、狠话版。

## V2 极简分享切换版

版本代号：`copy-v2-minimal-share-arrows`

修改日期：2026-06-04

修改前基线：`5e3a765 Copy V1 open mic mouthpiece`

修改目标：

- 删除左侧舞台中的“回答气泡”UI，降低信息拥挤感。
- 删除右侧类型标签中的“有梗概括”表述，让文案更少功能说明味。
- 顶部主操作收敛为单一“分享”入口。
- 分享文案池从 tab 切换改成左右箭头切换，让它更像可浏览文案轮播。

修改范围：

- 左侧舞台：移除回答气泡标题与台词胶囊，仅保留剧目切换控制。
- 顶部操作区：删除双按钮，保留一个“分享”按钮。
- 右侧类型模块：`类型描述 · 有梗概括` 改为 `类型描述`。
- 分享文案池：移除四个版本按钮，新增左右箭头、当前版本名和序号。
- 动效脚本：移除回答气泡相关更新逻辑，新增分享文案轮播状态。

保留原则：

- 保留场景云、人物/名人演员、剧目切换和右侧自评/点评。
- 保留 V1 的文案基调，不在本版继续重写核心文案。
- 不改变名人参剧的同步关系。

回退方式：

```bash
git checkout copy-v2-minimal-share-arrows
```

如需回到本版之前，回退到 V1：

```bash
git checkout copy-v1-open-mic-mouthpiece
```

验证方式：

- 页面不再出现 `回答气泡` 文案和回答胶囊。
- 页面不再出现 `有梗概括` 表述。
- 顶部只出现一个 `分享` 操作。
- 分享文案可通过左右箭头在四个版本之间切换。
- 截图保存 `screenshots/report-share-page.png`。

## V3 名人类型样本版

版本代号：`copy-v3-celebrity-type-samples`

修改日期：2026-06-14

修改前基线：`94a7ee2 Copy V2 minimal share arrows`

修改目标：

- 将名人切换后的右侧模块从“名人点评”改为“名人类型”。
- 降低“名人在评价用户”的感觉，改成“这个名人是同类型人格的戏剧化样本”。
- 让罗永浩和苏格拉底分别承担赛博判官的两种分支：行动版 / 追问版。

修改范围：

- 罗永浩幕：`名人点评 / 罗永浩说` 改为 `名人类型 / 罗永浩型赛博判官`。
- 苏格拉底幕：`名人点评 / 苏格拉底说` 改为 `名人类型 / 苏格拉底型赛博判官`。
- 两段正文从拟态点评改为类型样本说明，强调“名场面如何体现这个类型”。

保留原则：

- 默认态仍保留 `赛博判官自评 / 赛博判官说`。
- 名人幕仍保留有趣、带梗、可分享的语气。
- 不写成真实背书，不让名人像在正式给用户下判断。

回退方式：

```bash
git checkout copy-v3-celebrity-type-samples
```

如需回到本版之前，回退到 V2：

```bash
git checkout copy-v2-minimal-share-arrows
```

验证方式：

- 切换到老罗幕时，右侧显示 `名人类型 / 罗永浩型赛博判官`。
- 切换到苏格拉底幕时，右侧显示 `名人类型 / 苏格拉底型赛博判官`。
- 页面不再出现 `名人点评`。
- 截图保存 `screenshots/report-share-page.png`。
