# 报告分享页文案版本记录

用途：记录每一批网站文案修改，保证每个文案版本都可以通过 Git commit / tag 回退、对比和复用。

## 记录格式

```text
版本名：
版本代号：
修改日期：
修改前基线：
修改目标：
修改范围：
保留原则：
回退方式：
验证方式：
```

## V1 开麦嘴替版

版本代号：`copy-v1-open-mic-mouthpiece`

修改日期：2026-06-04

修改前基线：`dac655a Add report share copy workbench`

修改目标：

- 保留“公正感、锋利感、自洽感、传播感”。
- 增强梗浓度、网感和群聊截图传播感。
- 让文案更像“赛博判官替我开麦”，而不是产品在解释人格。

修改范围：

- 页面浏览器标题与描述。
- 页面顶部副标题和主操作按钮。
- 左侧舞台的默认态、罗永浩幕、苏格拉底幕：场景云、问句、回答气泡。
- 右侧类型说明、赛博判官自评、名人点评、短判词。
- 分享文案池和版本按钮。
- 拉绳、切幕、复制结果等微交互文案。

保留原则：

- 不改变页面结构。
- 不改变“场景云 = 出题 / 演员 = 演出 / 回答气泡 = 说台词”的核心模型。
- 不新增真实背书语气，名人内容仍保持拟态点评和戏剧化解释。
- 不把赛博判官写成单纯攻击型角色，仍保留“标准、证据、定义”的正当性。

回退方式：

```bash
git checkout copy-v1-open-mic-mouthpiece
```

如需回到本版之前，回退到修改前基线 commit：

```bash
git checkout dac655a
```

验证方式：

- 打开报告分享页确认默认态、罗永浩幕、苏格拉底幕文案同步切换。
- 截图保存 `screenshots/report-share-page.png`。
- 检查分享文案按钮是否能切换为标准版、开庭版、名人版、狠话版。
