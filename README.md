# NBTI Report Share Page

这是一个独立静态前端页面，聚焦 NBTI 报告分享页，不依赖 `NBTI_mwb` 或其他 demo 的运行时。

## 打开方式

直接用浏览器打开仓库根目录的 `index.html`：

```text
index.html
```
